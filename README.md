The limited compiler for C language.
------

### Vaild tokens
- **Keywords**: int char return if else while break
- **Operators**: = ! + - * / == != < > <= >= && ||
- **Special Symbols**: [ ] { } ( ) ; ,
- **Identifier**: [A-Za-z\_][A-Za-z0-9\_]\*
- **Number**: [0-9]+
- **Char**: '[.|\\n|\\t| ]'
- **Comment**: //

### Coding Style
Must contain at least one space between two tokens.
